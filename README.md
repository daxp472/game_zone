## ðŸ“– How to Run

1. Clone the repository.  
   ```bash
   git clone <repo-link>
   cd GameZone
    ```

2. Install dependencies.
    ```bash
   cd backend
   npm install
   cd ../frontend
   npm install
    ```

3. Start the server.
    ```bash
    cd backend
    npm start
    ```

4. Start the client.
    ```bash
    cd frontend
    npm start
    ```
